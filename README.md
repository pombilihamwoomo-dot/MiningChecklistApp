Group6-ChecklistApp

Overview
A mobile app for mining safety checks. Built with Expo + React Native and Firebase, it digitizes pre-shift equipment checklists, stores reports in Firestore, and lets supervisors monitor submissions in real time. Improves safety, accountability, and record-keeping.

Features
- Digital equipment checklist submission
- Real-time supervisor monitoring
- Cloud storage with Firebase Firestore
- User-friendly mobile interface

Installation
1. Clone the repo: https://github.com/pombilihamwoomo-dot/Group6-ChecklistApp/blob/main/README.md
2. Install dependencies: npm install
3. Run the app: expo start

Usage
- Workers log in and complete safety checklists before shifts.
- Supervisors view submissions in real time.
- Reports are stored securely for auditing.

Tech Stack
- React Native + Expo
- Firebase Authentication & Firestore
- GitHub for collaboration
